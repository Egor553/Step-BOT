# План внедрения: Бот и Mini App (Timeweb + Vercel)

## Текущее состояние
- **Фронтенд:** Развернут на Vercel (https://admin-topaz-seven.vercel.app). Версия v1.2.
- **Бэкенд:** Развернут на Timeweb (85.239.49.201).
- **Авторизация:** Гибридная (Авто по ID + ручной ввод кода).

## Файлы для деплоя
Все актуальные файлы для Telegram-бота и API собраны в папке `/deploy`. 
Для выгрузки на Timeweb используйте содержимое этой папки.

## Что сделано:
1. **Админка в боте:** Команда `/admin admin123` открывает меню выгрузки CSV (юзеры, идеи, запросы) и общую статику.
2. **Гибридная авторизация:** Mini App сначала пробует зайти сам, если не выходит — просит код. Бот выдает код при `/start`.
3. **Удаление старой статики:** Из Mini App убрана вкладка Stats для обычных юзеров.

## Инструкция по обновлению:
1. Очистить кеш в Telegram (Настройки -> Данные и память -> Использование памяти -> Очистить кеш).
2. Заменить файлы в папке бота на сервере Timeweb файлами из `/deploy`.
3. Перезапустить бота (`pm2 restart shag-bot`).
